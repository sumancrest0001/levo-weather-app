# Todo List App
  This is a n assignment for a software position at levo. It helps you to find current weather condition and 3 hour interval weather updates based on your current location.

## This project has this characteristics:
  - Shows your current location and weather conditions.
  - shows three hours interval weather


## Built With

- HTML, CSS, tailwind
- React, Next

## Getting started
- Clone the project using <git clone https://github.com/sumancrest0001/levo-weather-app>
- Run <cd levo-weather-app> from the current directory.
- Run <npm install> to have all the needed packages to run the app.
- Run <npm run start> command to run locally
